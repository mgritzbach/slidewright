# Fixture provenance

`slidewright-mit-template.pptx` is an original synthetic fixture authored and saved by Microsoft PowerPoint for Windows through `create-fixture.ps1`. The creation script then replaces workstation Office author fields with the neutral value `Slidewright contributors`; it does not alter presentation content. The fixture contains no third-party template, media, personal data, embedded font, or proprietary brand asset.

The committed PPTX is the canonical golden input. The script documents its construction but is not used by the edit or preservation auditor. The fixture is licensed under the adjacent MIT license.
